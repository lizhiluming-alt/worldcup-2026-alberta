# 世界杯观赛时间表 · Alberta

A single-file static web app showing the FIFA World Cup 2026 schedule in **Alberta time
(America/Edmonton)** with live scores, finished results, goals + scorers, betting odds,
and group standings.

- **Live & finished scores**, auto-refreshing
- **Goals**: minute, scorer, assist, type (header / penalty / own goal)
- **Odds**: 3-way decimal/European (win / draw / lose)
- **Standings**: all 12 groups (A–L)
- No backend — the browser fetches public ESPN endpoints directly

## Use it

Open `index.html` in any browser, or visit the GitHub Pages URL.

Data source: ESPN public APIs. Times shown in America/Edmonton.
